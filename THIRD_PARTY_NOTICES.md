# Third-Party Notices

This is the attribution appendix to [`THIRD_PARTY_LICENSES.md`](./THIRD_PARTY_LICENSES.md).

The summary document says which licenses are involved. This one reproduces, for every package
in the conservative **production closure** (backend runtime dependencies, frontend bundle
inputs, and their transitive dependencies), the license text and any `NOTICE` file the package
itself ships. Optional packages resolved by the lockfile can make this appendix larger than the
Linux image's package directory, never smaller. That is what MIT, BSD, ISC, Apache-2.0 and OFL each
require a redistributor to pass on: not only the license identifier, but the package's own
copyright line and terms when the package supplies them.

This file is copied into the local verification image. Generated by `bun run licenses:generate`; do
not edit by hand.

This appendix covers **224** third-party package versions in the conservative production closure.

## Packages shipping no license file

These packages declare a license but ship no LICENSE file of their own, so there is no
copyright line to reproduce. Their terms are the standard text of the declared license,
reproduced in the summary document; the copyright holder is the package author.
